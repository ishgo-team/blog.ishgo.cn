 Hexo静态博客自定义博客

Hexo 是一个基于 Node.js 的静态博客程序, 可以方便的生成静态网页托管在 github 和 Heroku 上。

为什么要用静态博客？

  • 跨平台同步文件
  • 系统简单易懂, 毫不臃肿, 方便改版
  • 众多第三方应用, 容易做自定义功能扩展
  • 纯静态, 极低的服务器压力
  • 最重要使用 Markdown 标记语法, 迁移什么的都非常方便
  • 能够托管在 Github 上
